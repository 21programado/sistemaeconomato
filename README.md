CONTROL DE ENTREGA Y DEVOLUCIÓN DE EQUIPOS

Esta carpeta contiene la aplicación para registrar la entrega y devolución
de equipos y materiales del servicio.

ARCHIVOS IMPORTANTES
--------------------
- equipos.py
  Es la aplicación principal. No borrar.

- registros.db
  Base de datos con todos los registros.
  No borrar ni mover. Por defecto se crea en AppData\Roaming >>Carpeta: ControlEquipos

- crear_acceso_escritorio.bat
  Sirve únicamente para volver a crear el acceso directo
  en el Escritorio si fue eliminado por error.
  Solo ejecutar si el acceso directo no está.

- Archivos PDF
  Son reportes generados por la aplicación.
  Se pueden conservar o borrar según necesidad.

USO NORMAL
----------
La aplicación se abre desde el acceso directo del Escritorio.
No es necesario abrir ninguno de estos archivos manualmente.

SI EL ACCESO DIRECTO DESAPARECE
-------------------------------
1. Abrir esta carpeta
2. Hacer doble clic en:
   crear_acceso_escritorio.bat
3. El acceso directo se volverá a crear automáticamente.

IMPORTANTE
----------
- No borrar archivos de esta carpeta.
- No mover archivos individualmente.
- Si es necesario copiar información, copiar la carpeta completa.

Ante dudas, consultar con el responsable del sistema.
Para obtener el .exe (v1.1) desde Releases puede hacerlo con la url: https://github.com/21programado/sistemaeconomato/releases/latest
